# 🚗 Cyber-Race 2026: Pro Physics Edition

Cyber-Race 2026 is a fast-paced 3D racing game built using Three.js.  
It features realistic driving physics, dynamic road generation, and advanced gameplay mechanics like nitro boost, obstacles, coins, and level-based challenges.

---

## 🎮 Features

- 🚗 Smooth car physics with realistic steering
- 🛣️ Procedural infinite road generation
- ⚡ Nitro boost system
- 💥 Collision & damage system
- 🪙 Coin collection system with animations
- 🌳 Environment with trees & grass
- 🌤️ Dynamic sky & lighting
- 🚧 Obstacles (stone blocks & ramps)
- 📊 HUD (Speed, Distance, Health, Nitro, Coins)

---

## 🆕 Advanced Gameplay (Levels)

- Level 3 → Traffic cars 🚗
- Level 5 → Police chase 🚓
- Level 7 → Night mode 🌙
- Level 10 → Boss obstacle 😈

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Three.js (3D Rendering)

---

## 🎯 Controls

- ⬆️ / W → Accelerate  
- ⬇️ / S → Brake / Reverse  
- ⬅️ / A → Turn Left  
- ➡️ / D → Turn Right  
- Shift → Nitro Boost  
- P → Pause  
- R → Restart  

---

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in browser
3. Enjoy the game 🎮
